# AI Job Market Analysis

## Overview
This project analyzes how Artificial Intelligence impacts different industries and job roles.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab
- Excel

## Key Insights
- Healthcare jobs are the most resilient to AI.
- Government roles show higher vulnerability.
- Higher education levels generally reduce automation risk.

## Visualizations
- Bar Charts
- Scatter Plots
- Heat Maps

## Dataset
AI Job Market Dataset
